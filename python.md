Variables
```python
# Dynamic typing
x = 100
random = 'Python'
```
Data Types
```python
# Integers
x = 4
# Floats
z = 3.1415
# Strings
name = 'Victor'
# Booleans
a = True
b = False
```

Casting -- Type Conversion
```python
current_year = "2020"
current_year = int(current_year)
current_year = str(current_year)
current_year = float(current_year)
```

Invalid Casting
```python
my_name = "Victor"
my_name = int(my_name) # => ValueError: invalid literal for int() with base 10: 'Victor'
```

Operators 
```python
a = 'Basic'
b = 'String'
a + b # BasicString

c = 'Victor'
c * 5 # => VictorVictorVictorVictorVictor
```

Reassinging values
```python
# Reassingning values
d = 18
d = d + 1 # => 19

e = 20
e += 1 # => 21
```
